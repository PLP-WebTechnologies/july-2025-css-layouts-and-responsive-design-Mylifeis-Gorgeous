Responsive web design

About

Responsive web design is about creating websites that are able to fit on all screen sizes

Tips

Just like the famous saying there are many ways of killing a rat, there are many ways of archeiving this in your designs. You could you use media querries where you define the dimensions that you want the changes to take place. i prefer using the width min and margin auto since i find it easy since it doesn't require a lot of things.

Importance

The importance of responsive design is that it enables all viewers of your website to have a wonderfull enjoyable experience while they are on you website. It also gives your website a professional look since it will look good on all devices irregadless of size

Fun Fact

When i was a starting on web development i used to wonder how to archive this. I could create websites that looked good on full screen but when i view them on small screen they would produce undesirable results. So when I learned about this it was trully a game changer
